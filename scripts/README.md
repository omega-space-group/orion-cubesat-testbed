# Scripts

Quick setup and automation scripts for the project.

## Available Scripts

- `setup.sh`: Initial project setup
- `build.sh`: Build all components
- `test.sh`: Run all tests
- `deploy.sh`: Deploy to hardware

## Usage

```bash
# First-time setup
./scripts/setup.sh

# Build everything
./scripts/build.sh

# Run tests
./scripts/test.sh
```
