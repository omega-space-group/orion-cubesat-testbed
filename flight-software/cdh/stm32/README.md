# C&DH - STM32 Implementation

C&DH implementation on STM32 Nucleo with FreeRTOS.

## Purpose

Flight-relevant implementation optimized for real-time performance, low power consumption and reliability.

## Hardware Setup

- **Board**: STM32 NUCLEO-H755ZI-Q
- **RTOS**: FreeRTOS
- **Interfaces**:
  - CAN: Onboard CAN peripheral
  - Flash: External flash for data storage
