# Payload - NVIDIA Jetson

AI inference and on-board processing using NVIDIA Jetson platform.

## Hardware

- **Platform**: NVIDIA Jetson (Xavier NX / AGX Orin)
- **OS**: Ubuntu 20.04 with JetPack SDK
- **GPU**: CUDA-enabled with Tensor Cores

## Software Stack

- **Deep Learning**: PyTorch, TensorFlow
- **Inference**: TensorRT for optimization
- **Vision**: OpenCV, CUDA-accelerated
- **ROS**: ROS 2 Humble integration
