# Hardware

Hardware documentation, configurations, and specifications.

## Contents

- **bus-configs/**: CAN and GigE configuration
- **schematics/**: Hardware schematics and diagrams
- **bom/**: Bill of Materials

## Hardware Overview

### Computers
- Raspberry Pi-4 8GB (C&DH prototype)
- STM32 NUCLEO-H755ZI-Q (C&DH production, EPS)
- NVIDIA Jetson Xavier NX / AGX ORIN (Payload)
- Xilinx KRIA SoC FPGA (Payload)

### Communication
- HackRF One (RF comms)
- CAN bus (critical comms)
- Gigabit Ethernet (high-bandwidth)

## Documentation

Each hardware component should be documented with:
- Specifications
- Pinout/connections
- Configuration
- Setup instructions
